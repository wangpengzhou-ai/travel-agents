# Style Presets

## `watercolor_postcard`

Refined watercolor travel postcard, ink-and-wash architecture or landscape, visible paper grain, airy brush edges, elegant travel sketchbook feeling without written text.

Sample: `assets/style_samples/watercolor-postcard-rome.png`

## `cinematic_landscape`

Cinematic travel landscape, realistic natural textures, high dynamic range, premium travel film still. Agent should remain a subtle small traveler.

Sample: `assets/style_samples/cinematic-landscape-sydney.png`

## `high_quality_3d_animation`

Original high-quality 3D animated movie look, rounded appealing forms, stylized buildings or landscapes, soft global illumination. Do not imitate a named studio, franchise, or existing character.

Sample: `assets/style_samples/3d-animation-new-york.png`

## `anime_travel`

High-quality Japanese anime travel-film background art, clean expressive linework, luminous color, soft atmospheric perspective, and detailed hand-painted landmarks. Use this when watercolor postcard needs a clearer stylistic contrast.

Sample: `assets/style_samples/anime-travel-guanajuato.png`

## Style Sample Labeling

The bundled sample images include a tiny handwritten city/date label for presentation. Daily wallpaper prompts should avoid readable text unless the user explicitly asks for this label treatment.
